Starting in 2010 through 2020, the City of Dublin, was the fastest growing city in California based on Census data. 
This analysis explores how the rapid population growth affected City, its impact on development, affordability, and how it compares to the County as a whole.
The Census data used for this analysis was DP05, B25070, and tl_2025_06_tract. 
A County_Boundary.shp file was used to create the Councty map and proprietary City Permit data with geolocations were used to create the development visualizations.
